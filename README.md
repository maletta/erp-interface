# Recriação da interface do discord

Foobar is a Python library for dealing with word pluralization.

## Instalação

Use o npm ou o yarn para instalar:

```bash
npm install
## ou
yarn install
```

## Executar projeto

Use o npm ou o yarn para executar:

```bash
npm run start
## ou
yarn start
```
